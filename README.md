# Personal Website

This repository contains a simple static personal website.

## Files

- `index.html` is the main page.
- `styles.css` contains the visual design.

## Run Locally

Open `index.html` in a browser.

## Customize

- Replace the placeholder text with your own bio.
- Update `mailto:hello@example.com` with your email address.
- Add real project names and descriptions in the Projects section.
